# Puzzle Game 数字华容道

## 简介

这是一个数字华容道游戏，一般称作"15Puzzle"（即4×4的华容道有15个块），但本项目包含了2×2到10×10一共9种难度；

比较有意思的有2点：如何判断打乱是否有解？如何配置颜色？详见：[功能介绍与技术细节](./docs/PUZZLE_GAME.md)

# Puzzle Game 现代贪吃蛇

## 简介

贪吃蛇也是一个非常经典的游戏，参考了现代俄罗斯方块相对于经典俄罗斯方块的新机制，在本项目中引入了2个新的机制：碰撞容错 + 锁定冲刺（Lock Dash）
